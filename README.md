# Low Level Shell
A simple shell built in C.
## Install

```
git clone https://github.com/low_level_shell
cd low_level_shell
make
./Bin/shell

```
#### To clean it up
```
make clear
```

## Configuration
You can configure your prompt by going to `[cmdPrompt](cmdPrompt.c)` file and changing `prompt` variable.

## Builtin Commands

we only have two builtin commands
- **exit**
- **env**
